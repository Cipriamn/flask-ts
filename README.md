# Flask TypeScript Project

This project is a full-stack application with a Flask backend and a TypeScript frontend. It is designed to provide a rental marketplace solution tailored for the Romanian market.

## Project Structure

```
flask-typescript-project
├── backend
│   ├── app
│   ├── config
│   ├── tests
│   ├── requirements.txt
│   ├── .env
│   └── README.md
├── frontend
│   ├── src
│   ├── public
│   ├── package.json
│   └── tsconfig.json
└── README.md
```

## Backend Setup

1. Navigate to the `backend` directory:
   ```
   cd backend
   ```

2. Install the required Python packages:
   ```
   pip install -r requirements.txt
   ```

3. Set up environment variables in the `.env` file.

4. Run the Flask application:
   ```
   flask run
   ```

## Frontend Setup

1. Navigate to the `frontend` directory:
   ```
   cd frontend
   ```

2. Install the required Node.js packages:
   ```
   npm install
   ```

3. Start the React application:
   ```
   npm start
   ```

## Usage

- The backend API will be available at `http://localhost:5000`.
- The frontend application will be available at `http://localhost:3000`.
